# rg_championship_2026_prediction

Проект по прогнозированию результатов Чемпионата России 2026 года на основе данных, которые были собраны вручную за 2023-2025 годы. Используя машинное обучение, было предсказано топ-5 участниц в категориях "юниорки" и "сениорки" с оценкой их вероятных баллов.

**Ссылка на проект:** [champ2026.ipynb](https://colab.research.google.com/drive/1hBpLIoNGIQpOhWDBqg7Ttp40NneoiZwr#scrollTo=2Ha0FHX8Tmrc)

---

## Как это сделано

**Технологии:** Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn

**Этапы работы:**

1. **Сбор данных:** Вручную собран датасет из 500+ записей о выступлениях гимнасток за 2023-2025 годы (соревнование, дата проведения, имя гимнастки,	год ее рождения,	регион,	разряд,	категория,	вид 1, вид 2, вид 3,	вид 4, сумма всех видов	, место).

2. **Исследовательский анализ (EDA):**
   - Визуализировала распределения результатов
   - Проанализировала корреляции между видами упражнений
   - Сравнила статистики юниорок и сениорок
   - Выявила ключевые различия между категориями

3. **Создание признаков (feature engineering):**
   Создала 7 осмысленных спортивных признаков:
   - **rest_days: дни отдыха** между соревнованиями
   - **trend3: тренд за 3 выступления** - динамика результатов
   - **record: личный рекорд** и **процент от рекорда** - насколько близко к лучшему результату
   - **season_consistency: стабильность в сезоне** - консистентность выступлений
   - **top3_frequency: частота попадания в топ-3** - как часто занимает призовые места
   - **consistent_growth: показатель роста** - баланс прогресса и стабильности

   Для обучения моделей сгенерировала 17 дополнительных признаков на основе истории выступлений каждой гимнастки.

4. **Моделирование:**
   - Сравнила три алгоритма: LinearRegression, RandomForest, XGBoost
   - Для каждой категории выбрала лучшую модель:
     - **Юниорки**: XGBoost (R^2 = 0.354, MAE = 2.824)
     - **Сениорки**: RandomForest (R^2 = 0.351, MAE = 3.228)
   - Обучила финальные модели на полных данных

5. **Прогноз и визуализация:**
   - Подготовила данные 40 активных юниорок и 33 активных сениорок
   - Сделала предсказания для ЧР 2026
   - Визуализировала топ-5 для каждой категории
   - Проанализировала важность признаков
