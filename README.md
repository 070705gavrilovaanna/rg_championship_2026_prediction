# rg_championship_2026_prediction

Прогнозирование результатов Чемпионата России 2026 по художественной гимнастике на основе исторических данных 2023-2025.

**Ссылка на проект:** [championship2026.ipynb](https://colab.research.google.com/drive/1qt3iVu2TiZEJSUgbSs3x9yZHriEIsJdb#scrollTo=kHtfGuQ_qA-H)

---

## Метрики

| Категория | Модель | R² | MAE | RMSE |
|-----------|--------|-----|-----|------|
| Юниорки | XGBoost | 0.354 | 2.82 | 3.96 |
| Сениорки | RandomForest | 0.351 | 3.23 | 3.79 |

**Бейзлайн (предсказание среднего балла):** R² = 0.00, MAE ~4.5

Модель объясняет 35% дисперсии результатов. Для спортивных данных (форма в день старта, судейство, травмы) это адекватный результат.

---

## Данные

- 341 запись о выступлениях (2023-2025)
- 103 гимнастки (49 юниорок, 57 сениорок)
- Ручной сбор данных

---

## Feature engineering

Создано 7 признаков:

| Признак | Смысл |
|---------|-------|
| rest_days | дни отдыха между соревнованиями |
| trend3 | динамика за 3 последних выступления |
| percentage_of_record | близость к личному рекорду |
| season_consistency | стабильность внутри сезона |
| top3_frequency | частота попадания в топ-3 |
| consistent_growth | прогресс * стабильность |

+ 17 лаговых признаков (3 прошлых выступления -> прогноз следующего).
