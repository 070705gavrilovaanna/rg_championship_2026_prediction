# Описание датасета

## Структура данных
- `competition` - название соревнования
- `competition_date` - дата проведения
- `name` - имя гимнастки
- `year_of_birth` - год рождения
- `region` - регион
- `rank` - разряд
- `category` - категория (юниорка/сениорка)
- `score1`, `score2`, `score3`, `score4` - оценки по видам
- `total` - сумма баллов
- `place` - занятое место

## Объем данных
- Период: 2023-2025 годы
- Количество записей: ~500
- Уникальных гимнасток: ~100
- Категории: юниорки (до 15 лет), сениорки (15+ лет)

## Источник
Данные собраны вручную из открытых источников о соревнованиях по художественной гимнастике.
